# @ffmpeg-wasm-browser/ffmpeg

The source code of @ffmpeg-wasm-browser/ffmpeg locates at **/packages/ffmpeg**.

## Development

```bash
$ npm run dev
```

## Build

Transpile Typescript to JavaScript.

```bash
$ npm run build
```

## Lint

```bash
$ npm run lint
```

## Publish

Simply run `npm publish` under **packages/ffmpeg**.
